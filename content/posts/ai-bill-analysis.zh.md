## 为什么你该翻翻自己的账单

说实话，大多数人对自己的钱去哪了，心里是没数的。

不是收入不够，是不知道钱怎么就没了。

每个月工资到账，花着花着就见底了。打开账单看一眼，密密麻麻几百条记录，眼睛一花就关了。月末一看余额——就这？

谁有空一条一条看？

但AI有空。

它不嫌烦，不走神，不会看到"奶茶"两个字就自动跳过。你把一年的流水丢给它，它能把每一笔消费分门别类，按月统计，甚至帮你揪出那些你自己都忘了的自动续费。

额，你真的确定你没有正在被悄悄扣款的会员？

## 手把手教你：三步搞定

整个过程不需要懂技术，不需要花钱，不需要梯子。十分钟搞定。

![操作流程](/images/illustration-2.png)

### 第一步：导出你的账单

**微信支付：**
打开微信 → 我 → 服务 → 钱包 → 账单 → 右上角"常见问题" → 下载账单 → 选用途"个人对账" → 选时间范围（建议选一整年） → 用PDF或CSV格式 → 发到你邮箱。

**支付宝：**
打开支付宝 → 我的 → 账单 → 右上角"..." → 开具交易流水证明 → 选时间范围 → 导出到邮箱。

有几个小tips：选择"个人对账"才可以选择一整年的时间，"证明材料"只能最长一次拉三个月的时间；邮件收到后打开附件可能需要输入密码，这个密码一般在申请记录的页面中会给到。

![](/images/IMG_4126.png)

![](/images/IMG_4124.png)两个都导，效果最好。因为你可能微信买外卖、支付宝交水电费，分开看永远看不到全貌。

导出之后去邮箱里把文件下载到手机或电脑上，准备下一步。

### 第二步：上传给AI

推荐用 **Kimi**（kimi.ai）。

原因很简单：国内直接用，不需要梯子，支持上传文件，而且对中文账单的理解能力很强。

打开Kimi，把你导出的PDF文件直接拖进对话框。如果是手机端，点附件按钮上传。

### 第三步：给它一句提示词

这是关键。你不用写一大段指令，一句话就够：

> "帮我按类别分组，按月统计，找出每月固定扣款的订阅服务。"
> 
![](/images/Pasted_image_20260315185825.png)
然后等几十秒。

它会返回给你三样东西：

- **消费分类表**：餐饮、交通、购物、娱乐、生活缴费……每个类别花了多少，占比多少
- **月度趋势**：哪个月花得最多，哪个月最省，波动在哪里
- **订阅列表**：每个月固定扣款的那些服务——视频会员、云空间、App订阅

![](/images/Pasted_image_20260315190133.png)

一年的账，几分钟翻完。以前要么花钱请会计，要么自己Excel一条条手动分类。现在一句话的事。

## 你可能会发现什么

我跟几个朋友聊了聊，他们用AI翻完账单之后，反应出奇一致。

沉默。

然后是："我怎么花了这么多？"

### 外卖占比惊人

月均1000到2000块花在外卖上的人，不在少数。一年下来，一两万。你以为只是"偶尔点一下"，AI告诉你——这是你最大的支出类别之一。

配送费、包装费、满减凑单多买的那些东西，加在一起比你想的多得多。

### 忘记取消的自动续费

这是最扎心的部分。

每个月几十块，有时候上百块——视频网站会员、音乐会员、某个云盘扩容、某个你下载过一次就再没打开过的App。它们安安静静地每个月扣一次，从不提醒你，从不打扰你。

一年加起来，几百到上千块。

你品品。

这些钱你可能完全没有意识到在花。不是因为金额大，恰恰因为金额小——小到你不会注意，但大到一年下来很可观。

### 深夜冲动消费

这个现在都是常态了。

把晚上11点以后的订单单独拉出来看一看。你可能会发现一个"夜间版的自己"——情绪上头就下单，第二天完全想不起来买了啥。

深夜消费往往是情绪驱动的。累了、烦了、无聊了、偶然刷到的直播间，随手就买了。单笔不大，但频率高。

这些钱，不翻账单你永远不知道。翻了，就再也装不了不知道。

## 进阶玩法

基础版翻完之后，你还可以继续问AI：

- "帮我找出金额最大的10笔非必要消费"
- "按时间段分析，看看周末和工作日的消费习惯有什么不同"
- "对比上半年和下半年，消费趋势是在涨还是在降"
- "帮我做一个每月预算建议，基于我的实际消费模式"
- "把所有外卖订单单独拉出来，算一下如果自己做饭能省多少"

AI不会judge你。它只负责把数字摊开来。

决定怎么调整的，还是你自己。

## 安全提醒——这条必须看

先说个严肃的事。

**账单是你最敏感的隐私数据之一。**

你的每一笔交易记录里，藏着你的生活轨迹、消费习惯、收入水平、活动范围，甚至社交关系。这些信息拼在一起，比你的朋友圈更真实地描述了"你是谁"。

所以在把账单交给AI之前，几件事要注意：

**不要随便传到不信任的平台。** 有些野鸡网站说"免费AI记账"——三思。你省下的那点时间，换走的可能是你所有的财务数据。

**用Kimi的话，记得关掉训练选项。** 设置里找到"允许用于模型训练"，关掉。你的账单不需要成为别人模型的养料。

**最安全的方式是用本地部署的AI。** 数据不出你的电脑，最踏实。但门槛也最高，适合有技术基础的人。

打个不太恰当的比方：账单就是你的财务裸照。你会随手发给陌生人吗？

对你的数据，负点责。

### 高阶版提示词，可以交给GPT/Claude/Gemini等大模型

```
您是一位资深个人财富分析师兼数据处理专家。您的任务是分析我跨越xxxx年、分布在三个平台（银行卡、微信支付、支付宝）的原始个人交易记录，清理重叠数据，并生成一份高度结构化、具有洞察力的财务报告。
  
背景与挑战:
- 时间段：xxxx年
- 数据源：银行对账单、微信支付、支付宝
- 核心挑战：这三个账户之间存在频繁的内部资金流转，需要去重。
  
处理要求:
1. 数据去重（关键）：识别所有“内部转账”（如银行卡转微信、支付宝提现到银行卡、信用卡还款）。将这些标记为“内部流转”，并在真实收支计算中剔除它们。
2. 真实收入识别：识别真正的外部收入来源（如薪资、投资回报、项目结算等）。
3. 价值导向分类：摒弃标准会计标签，将支出分为以下战略类别：硬性财务承诺（房贷、水电、保险）、家庭与核心圈（家用、育儿）、自我投资与健康（健身、效率软件）、资产配置（投资、储蓄）、生活方式与弹性消费（餐饮、娱乐）。
4. 异常检测：标记任何异常大额交易或不规律的消费模式。
5. 拒绝幻觉：如果某笔交易的用途完全不清晰，请将其标记为“未分类-待人工审核”并向我提问，切勿猜测。

交付物:

□ 统一去重后的全量交易主表。
□ 年度真实总收入与总支出对比分析。
□ 按上述“价值导向类别”划分的支出明细。
□ 内部流转资金及定期订阅的清单。
```
