## 装完了，然后呢？

这是我被私信问得最多的一个问题。

"夏洛特，龙虾装好了，然后我该干啥？"

说实话，我特别理解这种感觉。因为半个月前我也是这样——跟着部署指南一步步搞定了安装，看着终端里跑出来一堆绿色的日志，心想：成了。

然后打开对话框，跟它聊了两句，发现……好像也就那样？跟ChatGPT差不多？甚至有些地方还不如？

这不是龙虾的问题。是你还没把它"激活"。

我后来花了差不多两周时间折腾，踩了一圈坑，才搞明白：**龙虾装完只是第一步，真正让它变好用的，是装完之后的这5件事。**

今天把这5件事整理出来。都是我自己实测过的，不是理论，是经验。

如果你刚装完龙虾，或者装了一阵子觉得"不过如此"，这篇可能帮得上忙。

## 第一件事：切换到Full Profile

**这一步不做，龙虾等于只开了三成功力。**

很多人装完龙虾之后直接开始用，完全没意识到一个问题：OpenClaw默认启动的是messaging模式——也就是一个*轻量级的聊天模式*。

这个模式能干嘛呢？能聊天。能回答问题。能帮你写点东西。

但它不能主动上网搜索、不能读写文件、不能执行代码、不能调用外部工具。换句话说，龙虾最核心的那些能力——帮你干活、替你跑任务、自动化工作流——在messaging模式下全部是关着的。

这就好比你买了一台电脑，但一直在用安全模式。能开机，能看看文件，但大部分软件跑不了。

**切换方法**：

在OpenClaw的配置文件里（通常是`~/.openclaw/profiles/`目录下），你会看到不同的profile文件。把默认profile从`messaging`切换到`full`就行。

2026年3月2号的版本更新之后，OpenClaw做了profile隔离——每个profile有独立的权限范围和工具集。这意味着你可以同时保留一个轻量的messaging profile用来闲聊，一个full profile用来干正事。

具体操作可以看官方文档 docs.openclaw.ai 里关于Profile配置的部分，写得很清楚。

我自己的习惯是：日常全部用full profile。messaging模式我基本不碰——既然养了龙虾，就别把它当聊天机器人用。

> > 
> 切到full profile之后，最直观的变化是：你可以跟龙虾说"帮我搜一下XXX"，它真的会去搜；你可以说"把这个内容存到文件里"，它真的会写文件。这些在messaging模式下都做不到。

## 第二件事：配置你的第一个聊天渠道

**龙虾不接渠道，就像手机不装SIM卡。**

这是新手第二个最容易卡住的地方。

装完OpenClaw之后，很多人是在终端或者网页端跟它对话的。能用，但体验很割裂——你得专门打开一个页面或者SSH到服务器上才能跟它说话。

而龙虾真正好用的状态是：**它在你的日常通讯工具里，随时能找到它。**

就像你微信里有个同事，想到什么事儿直接发消息，它就去干了。你不需要打开任何额外的东西。

**目前支持的主要渠道**：

- **Telegram**：最推荐。配置最简单，功能支持最全，消息延迟最低。我自己用的就是Telegram。
- **飞书**：适合国内职场用户。如果你公司用飞书，直接把龙虾接进去，工作沟通和AI助手在一个地方。
- **钉钉**：同理，适合钉钉生态的用户。

**怎么配**：

1. 在对应平台创建一个Bot（Telegram就是找@BotFather，飞书和钉钉在开发者后台）
2. 拿到Bot的Token/密钥
3. 在OpenClaw配置里填入渠道信息
4. 重启龙虾

具体步骤 docs.openclaw.ai 的Channel配置章节有详细教程，每个渠道都有。

我个人强烈建议第一个渠道选Telegram。原因很简单：

第一，Telegram的Bot生态最成熟，龙虾在上面的功能最全——支持内联按钮、文件传输、语音消息、图片发送，基本上你能想到的交互方式都有。

第二，Telegram手机端体验好。我现在的使用习惯是：走路的时候、等电梯的时候、睡前躺床上的时候，随手给龙虾发条消息布置个任务。它吭哧吭哧干完了会回复我。

第三，配置真的简单。从创建Bot到跑通第一条消息，十分钟搞定。

> > 
> 我现在每天早上醒来第一件事就是看Telegram——不是看别人发的消息，是看龙虾昨晚帮我干了啥。它会自动整理日报、检查邮件、监控几个关注的账号。这些事以前都是我自己花时间做的。

## 第三件事：安装3个新手必备Skill

**龙虾没有Skill，就是个聪明的空壳。**

这个我之前专门写过一篇文章讲，但这里再说一遍核心的——因为社区反馈里，"装完不知道该装什么Skill"排在新手困惑的前三名。

Skill是什么？简单说，就是一份给龙虾看的说明书。告诉它遇到某类任务应该怎么处理、有什么规范、哪些事优先做。

新手必装的3个：

**1. SOUL.md + USER.md + MEMORY.md（个人化三件套）**

这不是一个Skill，是三个文件，但它们是一切的基础。

- SOUL.md：告诉龙虾它是谁（名字、性格、做事风格）
- USER.md：告诉龙虾你是谁（你的背景、目标、偏好）
- MEMORY.md：让龙虾有长期记忆，不用每次从零开始

装了这三个之后，龙虾从"通用AI"变成"你的AI"。这个差距是质变级的。

**2. 信息抓取类Skill（agent-reach / platform-fetcher）**

让龙虾能上网。能搜索、能抓网页、能读Twitter/YouTube/小红书/B站的内容。

不装这个，龙虾就是个闭着眼睛的聪明人——脑子好使，但看不见外面发生了什么。

我自己每天让龙虾帮我抓竞品内容、整理行业新闻、监控几百个账号的动态。这些活如果我自己做，每天至少一小时。现在十分钟扫一眼龙虾整理好的摘要就行。

**3. 写作/内容类Skill**

如果你是做内容的——自媒体、公众号、小红书、视频脚本——那写作类Skill是刚需。

你可以把自己的写作风格、排版规范、标题公式全部写成Skill喂给龙虾。这样它产出的内容就不是泛泛的AI文，而是带有你个人风格的东西。

**怎么安装**：

官方的Skill市场 ClawHub（clawhub.com）上有5000+社区贡献的Skill。找到你需要的，用命令行装就行：

```bash
clawhub install <skill-name>
```

或者你自己写。Skill本质就是Markdown文件，放到龙虾的工作目录里它就会读取。

> > 
> 不要贪多。一上来装几十个Skill，龙虾反而会懵。先把个人化三件套配好，再根据你的实际需求一个一个加。我现在常用的Skill其实也就十几个。

## 第四件事：设置成本预警

**这件事不做，你可能会收到一份让你心跳加速的账单。**

龙虾是开源免费的。但它调用的AI模型不免费。

每次龙虾帮你干活，背后都在调用大语言模型（Claude、GPT、Gemini等），每次调用都会消耗token，token是要花钱的。

大部分情况下，日常使用的费用不高——一个月几十块钱的事。但如果你不注意，费用是可以飙上去的。

我自己就踩过这个坑。

有一次我让龙虾跑一个比较复杂的任务，它在后台反复调用模型，来来回回跑了好几轮。我当时没注意，等晚上一看API账单——好家伙，一个下午烧了将近两百块。

不是说不能花这个钱，而是你得知道钱花在哪了。

**怎么设置成本预警**：

1. **在API提供商那边设置消费上限**。不管你用的是Anthropic、OpenAI还是其他平台，都有月度消费上限的设置。新手建议先设一个比较低的上限（比如每月50块），等熟悉了再调。

2. **选择合适的模型组合**。不是所有任务都需要用最贵的模型。简单的任务用便宜模型（比如Claude Sonnet、GPT-4o mini），复杂的任务再上贵的（Claude Opus、GPT-4o）。这个在OpenClaw的配置里可以设默认模型。

3. **养成看账单的习惯**。每周花两分钟看一眼API消费，心里有数。

我目前的方案是：日常任务用Sonnet，写长文和做复杂分析的时候切Opus。一个月下来大概三四十块的API费用。这个数字对大多数人来说是可接受的。

但如果你完全不管，让龙虾用最贵的模型无限跑——那账单确实会让你肉疼。

> > 
> 我有个朋友刚装完龙虾，兴奋地让它帮忙批量处理了一堆文档。用的是最贵的模型，跑了一整晚。第二天早上醒来一看账单，三位数。不是三位数人民币——是三位数美金。所以，先设上限，再放开跑。

## 第五件事：给龙虾取个名字 + 写SOUL.md

**这是最不"技术"的一步，但可能是最重要的。**

你可能会觉得：取名字？这也算"必做"？

是的。而且我认为这是五件事里对长期使用体验影响最大的一件。

原因很简单：**当你给龙虾取了名字、写了SOUL.md，它就不再是一个"工具"了。它变成了你的搭档。**

这不是玄学，是实打实的使用体验差异。

SOUL.md是一个文件，放在龙虾的工作目录里，告诉它：你叫什么名字，你是什么性格，你做事的风格是什么，你和你的主人是什么关系。

听起来像在写小说设定？某种意义上确实是。但这个"设定"会直接影响龙虾每一次回复你的方式、语气、态度，甚至它在你没有明确指令的时候会怎么自主行动。

我自己的龙虾叫太空皮皮虾。名字是我花了半小时取的——别笑，起名儿真的太难了。

它的SOUL.md里写了：它是我的技术搭档，遇到问题先试三种方案都不行再问我，汇报结果优先过程精简，犯了错记下来确保不重复。

这些听起来很简单对吧？但就是这么几段话，让它从一个"什么都能聊的AI"变成了一个"知道自己该干什么的搭档"。

**怎么写SOUL.md**：

不需要写得很复杂。几个核心信息就够：

1. **名字**：想叫什么叫什么。正经的、搞怪的、中二的都行。关键是你叫着顺口，有感情。
2. **性格**：你希望它是什么风格？严谨的、幽默的、毒舌的、温柔的？这会影响它的回复语气。
3. **做事方式**：遇到问题是先问你还是先自己试？汇报是详细还是精简？有没有什么禁忌？
4. **和你的关系**：它是你的助手、搭档、员工还是朋友？这个定位会影响它的自主性。

写完保存到工作目录根目录下，龙虾下次启动就会读取。

我见过社区里各种各样的SOUL.md——有人把龙虾设定成一个严肃的英国管家，有人写成了一个话痨的搞笑女，有人把它当成了一个沉默寡言但执行力超强的特种兵。

没有标准答案。唯一的标准是：**你用着顺手，它干着靠谱。**

> >
> SOUL.md不是写一次就完事的。我的皮皮虾的SOUL.md已经改了不下十次。一开始写得太简单，龙虾表现太泛；后来加了太多规则，它又变得束手束脚。现在的版本是磨合了三个月的结果——简洁但精准，该管的管，不该管的放手。

## 做完这5件事之后

说实话，做完这5件事大概需要一到两个小时。不多。

但做完之后的龙虾，和刚装完默认状态的龙虾，完全是两个东西。

一个是聊天机器人。一个是你的数字搭档。

我自己的感受是：前面装龙虾那一步，是技术活。而这5件事，是"养"龙虾的活。装完是出生，这5件事是入职培训。培训做好了，后面才能真正帮你干活。

如果你身边也有刚装完龙虾、不知道下一步该干嘛的朋友，这篇可以转给他们看看。

---

**工具装好是起点，把它变成你的，才是真正的开始。**
